## Technical SEO for Local E-commerce

### Local Schema Markup:

#### E-commerce Local Business Schema:

**Product + Local Business Schema:**
```json
{
  "@context": "https://schema.org",
  "@type": ["LocalBusiness", "OnlineStore"],
  "name": "[Business Name] - Orange County",
  "description": "Orange County e-commerce business specializing in [products]",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "[OC City]",
    "addressRegion": "CA",
    "postalCode": "[ZIP]",
    "addressCountry": "US"
  },
  "areaServed": {
    "@type": "GeoCircle",
    "geoMidpoint": {
      "@type": "GeoCoordinates",
      "latitude": 33.7175,
      "longitude": -117.8311
    },
    "geoRadius": "50000"
  },
  "makesOffer": [{
    "@type": "Offer",
    "itemOffered": {
      "@type": "Product",
      "name": "[Product Name]",
      "description": "[Product Description] - Available in Orange County",
      "offers": {
        "@type": "Offer",
        "availability": "https://schema.org/InStock",
        "price": "[Price]",
        "priceCurrency": "USD",
        "availableAtOrFrom": {
          "@type": "Place",
          "name": "[OC Store Location]"
        }
      }
    }
  }]
}
```

**OC-Specific Schema Properties:**
- `areaServed`: Orange County geographic area
- `availableAtOrFrom`: Specific OC store locations
- `pickupAvailable`: True for OC locations
- `deliveryAvailable`: True for OC service area
