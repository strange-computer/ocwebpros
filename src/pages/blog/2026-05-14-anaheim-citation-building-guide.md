## The Wrong Way: Common Anaheim Citation Mistakes

### Mistake #1: Inconsistent NAP (Name, Address, Phone)

**Wrong:**
- Google: "123 Main St, Anaheim, CA"
- Yelp: "123 Main Street, Anaheim, CA 92805"
- Chamber: "123 Main St., Anaheim, California"
- Yellow Pages: "123 Main, Anaheim, CA"

**Right:**
- Everywhere: "123 Main Street, Anaheim, CA 92805"
- Exact same formatting
- Exact same business name
- Exact same phone number

### Mistake #2: Missing Anaheim-Specific Details

**Wrong:**
- Generic business description
- No Anaheim references
- No Disneyland/Convention Center mentions
- No neighborhood specification

**Right:**
- "Anaheim-based business serving Disneyland area since 2010"
- "Located minutes from Anaheim Convention Center"
- "Serving Anaheim Hills and surrounding communities"
- "Trusted by Anaheim residents and visitors"

### Mistake #3: Incomplete Directory Profiles

**Wrong:**
- Basic information only
- Missing categories
- No photos
- Incomplete service descriptions

**Right:**
- Complete business profile
- Multiple relevant categories
- 10+ quality photos
- Detailed service descriptions
- Operating hours
- Payment methods
- Service areas

### Mistake #4: Ignoring Seasonality

**Wrong:**
- Static information year-round
- No seasonal updates
- Missing event-related information

**Right:**
- Seasonal hour updates
- Event-specific information
- Convention season specials
- Holiday season updates

### Mistake #5: One-Time Setup

**Wrong:**

**Right:**
- Regular profile updates

- Seasonal information updates
- Monitor for new Anaheim directories

## Anaheim NAP Consistency Checklist

### Business Name Consistency:
- [ ] Exact same business name everywhere
- [ ] No abbreviations unless consistent
- [ ] No extra descriptors unless consistent
- [ ] Legal name vs DBA consistent

### Address Consistency:
- [ ] Street: "123 Main Street" (not "123 Main St.")
- [ ] City: "Anaheim" (not "Anaheim, CA" unless consistent)
- [ ] State: "CA" (or "California" if consistent everywhere)
- [ ] ZIP: "92805" (full 5-digit, no extensions unless consistent)
- [ ] Suite/Unit: Include if present, consistent formatting

### Phone Consistency:
- [ ] Same phone number everywhere
- [ ] Format: "(714) 555-1234" (consistent formatting)
- [ ] No tracking numbers unless managed properly
- [ ] Toll-free vs local consistent

### Website Consistency:
- [ ] Exact same URL everywhere
- [ ] No www vs non-www inconsistencies
- [ ] No HTTP vs HTTPS inconsistencies
- [ ] No tracking parameters unless managed
